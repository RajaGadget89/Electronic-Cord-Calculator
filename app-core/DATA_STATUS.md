# สถานะข้อมูล & Engine — Milestone 1 ✅ เสร็จสมบูรณ์

**แหล่งข้อมูล (ยืนยัน):** ภาคผนวก A และ บทที่ 9 ของ *คู่มือการติดตั้งระบบไฟฟ้าอย่างมืออาชีพ (EIT Standard 2564)* โดย ลือชัย ทองนิล (ประธานสาขาไฟฟ้า วสท.) ซึ่งพิมพ์ตารางจากมาตรฐาน **วสท. พ.ศ. 2564** โดยตรง

## ครบและตรวจสอบแล้วทั้งหมด
- **Calculation engine** (`src/calc.ts`) — ครบ 8 ขั้นตอน วสท. + กฎ "เบรกเกอร์ปกป้องสาย" + ตรวจแรงดันตก (เลื่อนขนาดสายอัตโนมัติถ้าเกิน 3%)
- **Unit tests ผ่าน 21/21** รวม **ตัวอย่างโจทย์จริงในมาตรฐาน 3 ข้อ**:
  - ตัวอย่าง 3.1: 1φ 12×200VA ร้อยท่อ → 2.5 ตร.มม. (21A), เบรกเกอร์ 16A
  - ตัวอย่างกลุ่มวงจร: In 16A, 3 วงจร Cg=0.7 → It 22.8A → 4 ตร.มม. (28A)
  - ตัวอย่าง 9.1 (แรงดันตก): สูตร mV/A/m → 4.56% ตรงกับหนังสือ
- **ตารางกระแส (ampacity) ครบทุกขนาด 1–50 ตร.มม.**
  - **5-20** ร้อยท่อ (กลุ่ม 1&2, แกนเดี่ยว/หลายแกน, 1φ/3φ) — THW, NYY ในท่อ
  - **5-21** เกาะผนัง (กลุ่ม 3) — VAF
  - **5-22** ในอากาศบนลูกถ้วย (กลุ่ม 4) — VCT, THW
  - **5-23** ฝังดิน (กลุ่ม 5&6, ฐาน 30°C) — NYY
- **ตารางแรงดันตก 9.1 (สายเดี่ยว) และ 9.2 (หลายแกน)** — mV/A/m ครบ 1–50 ตร.มม.
- **ตัวคูณ derate:** อุณหภูมิ 5-43 (อากาศ) + 5-44 (ฝังดิน), กลุ่มวงจร 5-8 — ครบ
- **พิกัดเบรกเกอร์, คุณสมบัติสาย, pf โหลด, scope guard 50 ตร.มม.** — ครบ

## ไม่มีค่าที่เดา
ทุกตัวเลขในตารางคัดจากเอกสาร วสท. 2564 โดยตรง และ cross-check กับตัวอย่างโจทย์ในมาตรฐาน (ค่า 1–6 ตร.มม. + ตัวอย่างแรงดันตกตรงกันทุกจุด)

## ข้อจำกัดที่ตั้งใจ (ตาม PRD)
- รองรับถึง 50 ตร.มม. เท่านั้น — เกินกว่านั้นแอปแนะนำให้ปรึกษาช่างผู้ชำนาญการ + Magnetic Contactor / Overload Relay / Phase Protection
- ตาราง 5-21 เก็บเฉพาะคอลัมน์ VAF (สายแบน PVC); NYY บนผนัง (กลุ่ม 3 กลม) ยังไม่ใส่เพราะ default ของ NYY คือฝังดิน — เพิ่มได้ภายหลังถ้าต้องการ
- แรงดันตกใช้ค่า mV/A/m แบบอนุรักษ์ (คอลัมน์ที่ให้ค่าสูงสุด) เพื่อความปลอดภัย

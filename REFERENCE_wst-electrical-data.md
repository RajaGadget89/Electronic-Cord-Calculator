# เอกสารข้อมูลอ้างอิงทางไฟฟ้า (มาตรฐาน วสท.) สำหรับ Calculation Engine

> รวบรวมจากการค้นคว้า — ใช้เป็นฐานข้อมูลของแอป
> **แหล่งหลัก (น่าเชื่อถือสูงสุด):** เอกสารบทที่ 5 "สายไฟฟ้าและการเดินสาย" โดย **นายลือชัย ทองนิล** (อดีตประธานสาขาวิศวกรรมไฟฟ้า วสท., กรรมการสภาวิศวกร, ประธานคณะอนุกรรมการปรับปรุงมาตรฐานการติดตั้งทางไฟฟ้าฯ) อ้างอิงมาตรฐานการติดตั้งทางไฟฟ้าสำหรับประเทศไทย และ มอก.11-2553
> **มาตรฐานฉบับที่ใช้:** มาตรฐานการติดตั้งทางไฟฟ้าสำหรับประเทศไทย ฉบับปรับปรุงล่าสุด (พ.ศ. 2564)

> ⚠️ **หมายเหตุความปลอดภัย:** ค่าตัวเลขในเอกสารนี้แบ่งเป็น **[ยืนยันแล้ว]** (คัดจากแหล่งหลักโดยตรง) และ **[ต้องคัดลอก+ตรวจสอบ]** (ต้องดึงจากเล่มมาตรฐาน วสท. 2564 ฉบับจริงใน Milestone 1 ก่อนเปิดใช้งาน) — จะไม่ใส่ตัวเลขที่เดาเอาเด็ดขาด

## 5. ขั้นตอนหาขนาดสาย + เบรกเกอร์ (ตามแหล่งหลัก) **[ยืนยันแล้ว]**

```
1. คำนวณกระแสโหลด Ib   (Ib = P/(V·pf) 1φ ; P/(√3·V·pf) 3φ)
2. เลือกเบรกเกอร์ In ≥ Ib  (จากพิกัดมาตรฐาน)
3. ต้องการขนาดกระแสสาย It โดย It ≥ In / (Ca × Cg)
4. เลือกขนาดสายจากตารางที่ค่าในตาราง ≥ It
5. ตรวจแรงดันตก ถ้าเกินเกณฑ์ → เลื่อนขนาดสายขึ้น
6. เงื่อนไขปกป้องสาย: In ≤ Iz(สายหลัง derate) เสมอ
```
> **หลักสำคัญ:** เบรกเกอร์ต้องปกป้องสาย → พิกัดกระแสสาย (หลัง derate) ต้อง ≥ พิกัดเบรกเกอร์
> พิกัดเบรกเกอร์ MCB มาตรฐาน: **6, 10, 16, 20, 25, 32, 40, 50, 63, 80, 100, 125 A**

## 6. แรงดันตก (Voltage Drop)
- **เกณฑ์:** วงจรย่อย **≤ 3%** (รวมทั้งระบบไม่ควรเกิน 5%)
- **สูตร:** 1φ: `Vd = 2·I·L·(R·cosθ + X·sinθ)` ; 3φ: `Vd = √3·I·L·(R·cosθ + X·sinθ)`
- **ค่า R, X (Ω/km) ต่อขนาดสาย** หรือค่า **mV/A/m** → **[ต้องคัดลอก+ตรวจสอบ]** จากภาคผนวกมาตรฐาน วสท./แคตตาล็อกผู้ผลิต (เช่น Thai Yazaki)

---

## 7. Scope Guard — จำกัดที่ 50 ตร.มม. (ตามเจตนาผู้ใช้)
เมื่อกระแสโหลดสูงจนต้องใช้สาย **> 50 ตร.มม.** หรือเข้าใกล้ขีดจำกัด แอปต้อง **แสดงคำเตือน** ว่า:
> "งานนี้มีขนาดโหลดสูง ควรปรึกษาช่างไฟฟ้าผู้ชำนาญการ และอาจต้องใช้อุปกรณ์เพิ่มเติมนอกเหนือจากเบรกเกอร์ เช่น แมกเนติกคอนแทกเตอร์ (Magnetic Contactor), รีเลย์ป้องกันโหลดเกิน (Overload Relay), และระบบป้องกันเฟส (Phase Protection)"

สอดคล้องเจตนาแอป: ส่งเสริมความรู้/ความปลอดภัยสำหรับผู้เริ่มต้นทำงานระบบไม่ซับซ้อน

## 9. แผนตรวจสอบ (Verification) ก่อนเปิดใช้ Engine
1. คัดลอกตารางที่ 5-20, 5-21, 5-22, 5-23 (ครบทุกขนาด 1–50) จากเล่มมาตรฐาน วสท. 2564 ฉบับจริง
2. คัดค่า R/X (หรือ mV/A/m) สำหรับคำนวณแรงดันตก
3. **Cross-check ทุกค่ากับอย่างน้อย 2 แหล่ง** (เล่ม วสท. + แคตตาล็อกผู้ผลิตสายที่อ้างอิง วสท.)
4. เขียน unit test เทียบผลกับ "ตัวอย่างโจทย์" ในเอกสารของลือชัย ทองนิล (เช่น 2.5mm²→21A, NYY 4×120 บนราง→ตามตัวอย่าง)
5. ให้ผู้มีความรู้ด้านไฟฟ้าทวนก่อน release
